# Fastable2 Generated App

## Project Type
This is a frontend-only React + Vite + Tailwind CSS application.
Data is persisted using localStorage or IndexedDB (no backend).

## Key Commands
- `npm run dev` — Start development server (port 5173)
- `npm run build` — Build for production
- `npx playwright test` — Run all E2E tests

## File Structure
- `src/` — Application source code
- `e2e/` — Playwright E2E tests
- `e2e/screenshots/` — Captured screenshots

## Rules
1. Every feature must have E2E tests
2. Tests must pass before considering a feature complete
3. Take screenshots of key UI states
4. Use TypeScript strictly
5. Use Tailwind CSS for styling
6. Use React Router for multi-page apps
7. Use localStorage/IndexedDB for data persistence
